Abstract

Detecting Stress Before It Peaks

CalmSense is a smart IoT system designed to monitor physiological signals related to anxiety — primarily heart rate and breathing — in real time. The system detects abnormal increases in the user's arousal level and triggers an immediate calming response.

Built on a Raspberry Pi with physiological sensors and a real-time software pipeline, the core idea is to enable early detection of stress or anxiety, and help the user return to calm through biofeedback or guided breathing exercises — all without human intervention.

Anomaly Detection

🌬️

Guided Breathing

Triggers an immediate guided breathing exercise when elevated arousal is detected.

📊

Data Pipeline

Collects, analyzes, and responds to data in real time on a Raspberry Pi.

⚙️

Personal Calibration

Calibrates to each user's physiological baseline for maximum accuracy.

📡

Edge Processing

All processing happens locally on the device — no cloud dependency required.

System Architecture

How It Works

The system integrates hardware and software on a Raspberry Pi. Sensors gather raw physiological data, the analysis module processes and interprets it, and the response mechanism fires automatically when a threshold is exceeded.
